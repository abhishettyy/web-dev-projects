# Etch-a-Sketch

A classic browser-based Etch-a-Sketch application built with HTML, CSS, and JavaScript. This project allows users to create pixel art on a dynamic grid, mimicking the functionality of the nostalgic toy.

[**Live Demo**](https://abhishettyy.github.io/Etch-a-Sketch/) ---

## Features

* **Customizable Grid Size**: Users can specify a grid size between 16x16 and 200x200.
* **Color Picker**: Choose any color to draw with using the built-in color picker.
* **Progressive Shading**: Each pass over a square increases its opacity by 10%, allowing for a shading effect.
* **Reset Functionality**: A "Reset" button clears the grid, allowing the user to start a new drawing.

---

## Technologies Used

* **HTML5**: For the basic structure and content of the application.
* **CSS3**: For styling the user interface and the drawing grid.
* **JavaScript (ES6)**: For all the client-side logic, including grid generation, event handling, and drawing functionality.

---

## How to Use

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/abhishettyy/Etch-a-Sketch.git](https://github.com/abhishettyy/Etch-a-Sketch.git)
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd Etch-a-Sketch
    ```
3.  **Open the `index.html` file in your browser.**

That's it! You can now start drawing.
