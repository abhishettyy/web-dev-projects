# JavaScript Stopwatch

A clean and simple stopwatch application built with vanilla JavaScript, HTML, and CSS. It features standard stopwatch controls and a sleek dark/light mode toggle.

[**Live Demo**](<https://abhishettyy.github.io/Stop-watch/>) ---

## Features

* **Start, Pause, and Resume**: Control the timer with intuitive buttons.
* **Reset**: Easily reset the timer back to zero.
* **Accurate Time Display**: The time is displayed in a standard `HH:MM:SS` format.
* **Dark/Light Theme**: Switch between a light and dark theme with a single click for comfortable viewing in any lighting condition.
* **Clean UI**: A minimalist, circular design that is easy on the eyes.

---

## Technologies Used

* **HTML5**: For the structure of the application.
* **CSS3**: For all styling, including the dark mode theme.
* **JavaScript (ES6)**: For the core stopwatch logic and theme toggling.
* **Font Awesome**: For the moon icon used in the theme switcher.

## How to Use

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/abhishettyy/Stop-Watch.git](https://github.com/abhishettyy/Stop-Watch.git)
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd Stop-Watch
    ```
3.  **Open the `index.html` file in your favorite browser.**

You can now use the stopwatch!
